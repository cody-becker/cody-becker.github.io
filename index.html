Magnificent 7 Financial Analysis

A full financial analysis of Apple, Microsoft, Google, Amazon, Meta, Nvidia, and Tesla using real SEC EDGAR 10-K filings. Built a custom Python pipeline to pull data into PostgreSQL, then analyzed net income, EPS, revenue, profit margin, R&D spending, and long term debt across each company.
